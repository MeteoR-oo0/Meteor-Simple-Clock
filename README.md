# Meteor Simple Clock v2.1

NZXT Kraken 用カスタム時計 & モニタリング UI です。  
NZXT CAM の Web Integration 機能を使って、Kraken の LCD に自由なデザインの時計を表示できます。

> **v2** では表示デザインの自由度が大幅に向上しました。ブレンドモード、背景オーバーレイ、個別スタイル設定、プリセット、設定ファイルの保存・読込みなど多数の新機能を追加しています。
>
> **v2.1** では色設定 UI を刷新（スウォッチ＋グラデーションバー編集）し、背景の切り替えクロスフェード、画像へのグラデーション重ね、ネオングロー強化など、見た目の調整まわりを大きく改善しました。

## 💻 使用方法

1. CAM の設定から「Web Integration」を有効化
2. 上記 URL を CAM に登録
3. 表示が反映されます
4. CAM 上で表示を右クリック → `Configure` からカスタマイズできます

## 🔧 機能一覧

### 時計・日付
- デジタル時計（時・分、12/24H 切り替え）
- コロンの点滅オン/オフ、形を四角・丸に切り替え
- 秒数表示（数字）のオン/オフ
- 日付表示（複数フォーマット切替、非表示可）

### 秒バー & 秒リング
- 秒バー（シークバー）：伸縮/常時表示、長さ・色・ブレンドモード調整
- 秒リング：円形 LCD の縁を周回する秒針。スタイル（塗りつぶし/ドット周回）、グラデーション、ブレンドモードに対応

### センサー表示
- CPU / GPU 温度、冷却水温（LIQ）
- CPU / GPU 負荷（%）
- CPU / GPU ファン回転数
- メモリ使用量 / 使用率
- 最大 3 つまで同時表示。ドラッグで並べ替え・表示/非表示切り替え可

### 天気
- 都市名で天気を取得（℃ / ℉ 切り替え）
- 気温・天気アイコン・都市名を表示

### カスタムテキスト
- 好きな文字を自由に配置

### デザイン・スタイル
- フォント 10 種から選択
- フォントウェイト（Light / Normal / Bold / Black）
- 文字色・文字スケール・テキストシャドウ
- **ネオングロー**：文字を発光させる効果（グロー色の個別指定可、コロンの点も対応）
- **ブレンドモード**（通常/乗算/スクリーン/オーバーレイ/差の絶対値など 11 種）  
  全体一括 + 各ブロック個別指定に対応
- **各ブロック個別スタイル**：ブロックごとにフォント・太さ・色・ブレンドを独立設定可
- **カラースウォッチ**：単色・グラデーションのプリセット色をワンタップで選択（文字色・リング・背景）

### 背景
- **背景タイプ**：単色 / グラデーションを切り替え。グラデーションは色 2 点・角度・位置をバー UI で直感的に編集
- 透明度スライダーで色レイヤーの濃さを調整
- **背景画像**：正方形クロップ UI（ドラッグ＋ズームで位置調整）  
  GIF アニメーション対応（アニメーションを保持したまま表示位置を調整）
- 画像の上にグラデーション / 単色を重ねて色のかぶせ（tint）として使用可能
- **ぼかし**：背景画像のみに適用（時計や UI はくっきり保持）
- 背景の切り替え時に**クロスフェード**（画像・色ともに滑らかに変化）
- **背景オーバーレイ効果** 6 種（スキャンライン / CRT / グリッド / ドット / ノイズ / ビネット）  
  強さスライダーで濃さを調整可
- 最近使った画像の履歴（サムネイルからワンタップで再適用）

### レイアウト
- ブロックの並べ替え（ドラッグ）・表示/非表示
- 各ブロックのサイズ・上下余白を個別調整
- レイアウト変更時のアニメーション（プレビュー時）

### プリセット & バックアップ
- **組み込みプリセット 9 種**（Default / Aurora / Ocean / Retro / Neon / Mono / Sunset / Terminal / Sakura）ワンタップで適用
- **設定のエクスポート/インポート**：全設定 + 背景画像を JSON ファイルで保存・復元
- 設定リセット
